# Dios Nuestro Padre - Aplicación para la Iglesia DNP

Esta es una aplicación desarrollada para la iglesia Dios Nuestro Padre (DNP), diseñada para proporcionar devocionales, fechas importantes y funcionalidades de perfil para sus usuarios. La aplicación está construida utilizando React Native y Express para el servidor.

## Características Principales

### Devocionales:
- Lista de devocionales con foto del autor, nombre y descripción.
- Al hacer clic en un devocional, se muestra un modal con su contenido completo.
- El modal cuenta con una flecha para volver y muestra el nombre del autor y el contenido del devocional.

### Fechas Importantes:
- Muestra una lista de las fechas más próximas con el día y el título correspondiente.
- Incluye un flayer centrado para resaltar la importancia de estas fechas.

### Perfil:
- Funcionalidad para registrarse e iniciar sesión.
- Al iniciar sesión, se guarda el token y la información del usuario en el contexto global utilizando React Context.
- Permite editar perfil, cambiar contraseña y cerrar sesión.
- Los datos de perfil se almacenan en AsyncStorage para funcionar sin conexión a internet.

## Estilo y Diseño

- El diseño se ha adaptado para una experiencia de usuario intuitiva.
- Se utiliza React Navigation para la navegación entre los distintos apartados.
- Se han implementado animaciones al cargar los devocionales y las fechas al hacer swipup.
- El diseño de la interfaz se ha optimizado para una fácil comprensión y navegación.

## Tecnologías Utilizadas

- React Native: Para el desarrollo de la interfaz de usuario móvil.
- Express: Para el desarrollo del servidor que maneja la lógica de backend.
- AsyncStorage: Para almacenar datos localmente en el dispositivo móvil.
- React Context: Para el manejo del estado global de la aplicación y almacenar la información de sesión del usuario.
- React Navigation: Para la navegación entre los distintos apartados de la aplicación.
